---
This project, written in Bash, allows users to search for the **most popular baby names** in the United States from the late 1880s to 2022, using data from the Social Security Administration. 
Key features include : displaying top names for any year bewteen 1880-2022, filtering by gender(female, male, both), customizing the number of names displayed (1-100), and providing comprehensive error handling and input validation.
